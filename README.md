## 🔒 Diretrizes de Engenharia e Segurança Aplicadas
Em minhas soluções e planejamentos de portfólio, adoto práticas estritas de conformidade auditadas por IA:
*   **Isolamento de Janelas (Anti-Tabnabbing):** Implementação obrigatória do atributo `rel="noopener noreferrer"` em hyperlinks externos para proteção contra vulnerabilidades de redirecionamento.
*   **Criptografia Nativa:** Especificação e uso obrigatório de tráfego seguro sob o protocolo HTTPS.
*   **Controle de Versão Limpo:** Fluxo de branches organizadas via Git para garantir um histórico rastreável e governança sobre o código de produção.
